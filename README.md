# 📻 Radio_Lnux

- Player de Rádio para Terminal

[![Shell Script](https://img.shields.io/badge/Shell_Script-✓-4EAA25?style=flat&logo=gnu-bash)](https://www.gnu.org/software/bash/)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![MPV](https://img.shields.io/badge/Requires-MPV-ff69b4.svg)](https://mpv.io/)

- Um player de rádio elegante para terminal Linux com estações pré-configuradas e interface visual estilizada.

## ✨ Características

- 🎵 **22 estações** de rádio pré-configuradas
- 🎨 Interface colorida e elegante no terminal
- 📊 Visualização de áudio simulada
- 🚀 Suporte a streaming ao vivo
- 🎮 Controles simples via teclado
- 🔄 Cache buffer para streaming estável

## 📋 Pré-requisitos

- Linux (qualquer distribuição)
- [MPV Player](https://mpv.io/) instalado


# Ubuntu/Debian
```
sudo apt install mpv
```
# Fedora/RHEL
```
sudo dnf install mpv
```
# Arch Linux
```
sudo pacman -S mpv
```

## 🚀 Instalação


# Clone o repositório
```
git clone https://github.com/bhu01hackerspace/Radio_Lnux.git
```

# Entre no diretório
```
cd Radio_Lnux
```

# Dê permissão de execução
```
chmod +x main.sh
```

# Execute
```
./main.sh
```

## 🎮 Como usar

1. Execute o script
2. Escolha o número da estação desejada (1-22)
3. Pressione CTRL+C para parar a transmissão e voltar ao menu
4. Digite 0 para sair
